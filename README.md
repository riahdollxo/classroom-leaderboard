## 📖 Overview

The Classroom Leaderboard was created to practice JavaScript, DOM manipulation, event handling, and responsive web design. Users can increase or decrease participant scores, making it useful as a classroom activity tracker, competition board, or learning exercise.

This project focuses on writing clean, readable code while creating an engaging and responsive user experience using only front-end technologies.

## ✨ Features

- 📈 Real-time score updates
- ➕ Increase participant scores
- ➖ Decrease participant scores
- 🖥️ Interactive leaderboard interface
- 📱 Responsive design
- 🎨 Clean and beginner-friendly code
- ⚡ No frameworks or external libraries

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript ES6

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/riahdollxo/classroom-leaderboard.git
```

### Navigate to the project folder

```bash
cd classroom-leaderboard
```

### Open the project

Open `index.html` in your preferred web browser.

No installation or dependencies are required.
